## Responsible Disclosure
- Please report security issues privately via email or GitHub security advisories.
- Do not disclose vulnerabilities publicly until they are resolved.

## Project Security
- RateRover prioritizes secure transaction execution (Gnosis Safe)
- Follows best practices for key management and agent isolation
- See [Architecture](../core-concepts/architecture) for details
